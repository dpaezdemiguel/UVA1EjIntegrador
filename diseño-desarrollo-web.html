Diseño y Desarrollo WEB

Sobre la materia

El Diseño y Desarrollo Web es una disciplina que abarca la planificación, creación y mantenimiento de sitios web y aplicaciones web. El diseño web se centra en la estética, usabilidad y experiencia del usuario, mientras que el desarrollo web se enfoca en la implementación técnica y la funcionalidad de los sitios.
